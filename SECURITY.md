# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in PoshKelvin, please report it
responsibly.

**Do not open a public issue for security vulnerabilities.**

Instead, please email [igoravl@gmail.com](mailto:igoravl@gmail.com) with:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Any potential impact you have identified.

You should receive an acknowledgment within 48 hours. The maintainer will work
with you to understand and address the issue before any public disclosure.

## Scope

This policy applies to the PoshKelvin module source code in this repository.
Issues in the upstream Kelvin API itself should be reported to
[Kelvin](https://kelvin.ai/) directly.
